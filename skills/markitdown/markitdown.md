# MarkItDown Conversion

Whenever you need to convert a URL or a local file to Markdown, use the `markitdown` tool.

This tool is particularly useful for:
- Converting web pages to clean Markdown.
- Converting local files like PDFs, Word documents, or Excel sheets to Markdown.

Prefer this tool over generic web searching or reading if the user provides a specific URL or file path that needs structured conversion.
